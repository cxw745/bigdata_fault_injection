# 大数据故障注入系统 - 原始配置文件与项目描述

## 文档概述

本文档记录了项目的原始配置文件、组件位置、故障类型清单和分布式架构信息，作为项目的基准参考文档。

## 第二部分：监控组件配置

### 2.1 Prometheus 监控配置

| 服务 | 端口 | 位置 | 监控目标 |
|------|------|------|----------|
| node_exporter | 9100 | 所有节点 | 系统指标（CPU/内存/磁盘/网络） |
| namenode_exporter | 9402 | cpf-1 | NameNode JMX 指标 |
| datanode_exporter | 9401 | cpf-2/3/4 | DataNode JMX 指标 |
| resourcemanager_exporter | 9403 | cpf-1 | ResourceManager JMX 指标 |
| nodemanager_exporter | 9404 | cpf-2/3/4 | NodeManager JMX 指标 |
| Prometheus Server | 9090 | cpf-1 | 数据存储和查询API |

### 2.2 Prometheus API 配置

PROMETHEUS_HOST = "http://localhost:9090"
PROMETHEUS_API = f"{PROMETHEUS_HOST}/api/v1"
